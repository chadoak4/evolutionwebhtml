The Evolution of the Web


The evolution of web design has come along way since the early days of the computer.
In the beginning, web design was very basic in form of design using a black screen with
only a few monochrome pixels in the year of 1989. In the later 1990’s web design began
to evolve. This was the era we were introduced to the web browser, JavaScript and Flash.
It was then that we began to question whether developers should learn design and designers
learn to code due to the small complications each of these presented. Since then,
technology is only becoming stronger and is allowing us to create a more friendly experience
for both the developers and the users. Technology is only going to continue to improve and
it’s important that both design and the program work together to perform the highest
quality end result for the user.
